## 快速开始
1. 安装 docsify-cli 工具
```
npm i docsify-cli -g
```
2. 如果想在项目的 `./docs` 目录里写文档，直接通过 init 初始化项目。
```
docsify init ./docs
```
3. 通过运行 `docsify serve` 启动一个本地服务器，可以方便地实时预览效果。默认访问地址 http://localhost:3000 。
```
docsify serve docs
```
